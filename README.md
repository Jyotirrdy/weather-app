# Weather App

A lightweight weather web app built with vanilla HTML, CSS, and JavaScript.

## Features

- Search weather by city name.
- Use your current location (browser geolocation).
- Shows current temperature, weather condition, wind speed, and today's high/low.
- Uses free Open-Meteo APIs (no API key required).

## Run locally

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000> in your browser.
